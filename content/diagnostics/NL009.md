### NL009: Upper bound of FOR should not be *OCC
The upper bound for FOR-loops should not be `*OCC`, as the length of the array will be recalculated on every iteration.

## Invalid

```natural
FOR #I = 1 TO *OCC(#ARR)
```

## Valid

```natural
#S-ARR := *OCC(#ARR)
FOR #I = 1 TO #S-ARR
```

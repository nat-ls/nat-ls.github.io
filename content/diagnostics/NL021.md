### NL021 Literal value truncated at runtime
The value of this literal is truncated at runtime. This will raise unexpected results.

## Non compliant

```natural
DEFINE DATA LOCAL
1 #VAR (A1)
END-DEFINE

#VAR := 'Hi'
WRITE #VAR
END
```

## Compliant

```natural
DEFINE DATA LOCAL
1 #VAR (A1)
END-DEFINE

#VAR := 'H'
WRITE #VAR
END
```

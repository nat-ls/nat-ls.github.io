### Description
Gitmarkers in comments can be confusing, remove any '<<<<<<<', '>>>>>>>' or '======='.

This behavior can be configured using the `natls.style.discourage_gitmarkers` option in the [analyzer configuration](/docs/analyzer-config.md)

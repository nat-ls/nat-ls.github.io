### Description
Natural does not allow independent variables to be groups.

## Invalid

```natural
INDEPENDENT
1 +GROUP
2 +VAR (A10)
2 +ANOTHER-VAR (A10)
```

## Valid

```natural
INDEPENDENT
1 +VAR (A10)
1 +ANOTHER-VAR (A10)
```
